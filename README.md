Sri’s Ride is a simple and responsive cab-booking website. It allows users to view cab services, check pricing, and contact the service provider easily. The aim of this project is to create a clean, modern, and user-friendly interface suitable for real service websites.

Features of Sri’s Ride:

Responsive and clean user interface

Home, Services, Pricing, and Contact pages

Service listing: city rides, hourly rentals, and long trips

Simple pricing section

Contact form for enquiries

Smooth navigation across all pages

Hosted online using InfinityFree


Technologies used: HTML, CSS, optional PHP, and InfinityFree hosting.

What I learned: Building responsive layouts, organizing web pages, creating good user experience, and deploying a project online.

Future improvements: Adding login, booking system, admin dashboard, payment gateway, and booking history.
